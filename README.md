# Encapsulación mediante Programación Orientada a Objetos

Este repositorio contiene la implementación de una simulación de una tortuga utilizando **Programación Orientada a Objetos (POO)** en Python, con el objetivo de aplicar el **principio de encapsulamiento**.

La simulación representa los movimientos de una tortuga usando **dibujo en texto (ASCII)** dentro de la consola, sin emplear librerías gráficas como `turtle`.

---

## Objetivo del Proyecto

- Refactorizar una versión funcional previa a una versión **orientada a objetos**
- Eliminar el uso de **variables globales**
- Encapsular el estado de la tortuga dentro de una **clase**
- Permitir la creación de **múltiples tortugas independientes**

---

## Requerimientos Funcionales

- Toda la lógica debe estar contenida en una clase `Tortuga`
- El estado de la tortuga (`posicion_x`) es un atributo de instancia
- Se prohíbe el uso de variables globales
- Se deben poder crear múltiples objetos sin que interfieran entre sí
- El movimiento se representa gráficamente con caracteres ASCII

---

## Conceptos Aplicados

- Programación Orientada a Objetos (POO)
- Encapsulamiento
- Clases y Objetos
- Atributos de instancia
- Métodos
- Separación de responsabilidades
- Simulación gráfica mediante texto
